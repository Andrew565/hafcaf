Name Type Attributes Description
id string The identifier to be used for this route.
linkLabel string <optional>
 What text to use when creating a menu item for this route. A menu item will not be created if a linkLabel is not provided.
linkTagClass string <optional>
 What css classnames to give to the menu item container for this route.
linkLabelClass string <optional>
 What css classnames to give to the actual link inside the menu item container for this route.
pageClass string <optional>
 What css classnames to give to the page for this route.
innerHTML string <optional>
 The content of the page. If not provided, will default to config.loadingHTML. Can be set or overwritten later using hafcaf.updateRoute().
onRender function <optional>
 A function which will be called each time this route is rendered (made active). Can include multiple functions within itself, if desired. When composing your onRender, keep in mind to take advantage of the hafcaf.listeners collection, which can be used to hold removeEventListener calls and other functions you would like to run when hafcaf switches away from this route.
